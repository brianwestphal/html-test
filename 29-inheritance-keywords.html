Inheritance keywords

Default on body: purple text, pink 3px border.

inherit: copies computed parent value (purple text, pink border).

initial: property's CSS initial value (black text, no border).

unset: inherits if inherited-prop, else initial (color → purple from parent; border → no border).

revert: rolls back to user-agent stylesheet (browser default).

revert-layer: rolls back to the previous layer — base green, not theme blue.
