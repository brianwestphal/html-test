Percentage resolution

Horizontal CB — % padding/margin against width

CB width 400px → 10% padding = 40px (top & bottom)
padding: 10% 0
CB width 400px → 10% margin = 40px
margin: 10% 0

Vertical writing mode — % padding/margin against the new INLINE size (=height)

In writing-mode: vertical-rl, the inline axis is vertical. % padding/margin still resolves against inline size, which is now the container's height. Visually the swelling happens on the horizontal axis of the box (block axis).

CB height 220px → 10% padding = 22px on the block axis
padding: 10% 0 (vert)

Auto-height parent — % height on child resolves to auto

height: 50% → auto (parent has no defined height)

Fixed-height parent — % height on child works

height: 50% of 120 = 60

min-height: 50% on auto-height parent — resolves to 0

min-height: 50% (no effect — parent auto)

min-height with content fallback: parent auto, child has content

min-height: 50% of 100 = 50, content larger wins
