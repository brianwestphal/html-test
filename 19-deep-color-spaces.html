color() & wide-gamut color spaces

Wide-gamut spaces (display-p3, rec2020, prophoto-rgb, a98-rgb, xyz-d65) only show their full range on capable displays. On sRGB displays they should fall back to the nearest sRGB equivalent.

Same hue across spaces

sRGB rgb(255,0,0)
color(srgb 1 0 0)
color(display-p3 1 0 0)
color(rec2020 1 0 0)
color(a98-rgb 1 0 0)
color(prophoto-rgb 1 0 0)

Linear vs gamma-encoded sRGB

srgb 50% gamma
srgb-linear 50%
srgb 21% gamma ≈ linear 50%

Fallback layering

If color() unsupported, expect the simpler color underneath via cascade.

red → p3 red
rgb green → rec2020 green
