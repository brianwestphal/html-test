list-style-image and list-style-position

list-style-image: url(…)

  • The marker is the orange SVG circle.
  • Marker repeats per list item.
  • Third item.

list-style-position: outside (default)

  • Square markers sit outside the content box, so wrapped text is flush with the first line's start and the marker sticks out into the padding.
  • Second item with a longer sentence that will wrap onto a second line to demonstrate marker vs content alignment.

list-style-position: inside

  • Square markers are inline with content, so wrapped text aligns beneath the marker rather than beneath the first character of content.
  • Second item with a longer sentence that will wrap onto a second line to demonstrate the difference.
