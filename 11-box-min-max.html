min/max sizing

max-width: 400px

Container is 100% wide but never wider than 400px (and never narrower than 150px).

min-width overrides shrinking: width 50 + min-width 200 = 200

width:50px but min-width:200px wins → box is 200px wide.

max-width overrides growth: width 900 + max-width 300 = 300

width:900px but max-width:300px wins.

height vs min-height (min wins if content is taller)

height:10, min-height:60 → min wins.
