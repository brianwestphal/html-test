Containing block resolution

An absolutely-positioned element's containing block is the nearest ancestor whose position is not static, OR any ancestor that establishes a CB for absolute descendants via transform / filter / will-change / contain:paint / perspective.

Percentage padding always against inline size

Both boxes use padding: 20% 0. In horizontal writing-mode the % is computed against width; in vertical-rl, against height (the new inline size). Padding swells in opposite axes accordingly.
