text-emphasis — east-asian style emphasis marks

Mark style

The dot emphasis is the default east-asian style.

You can use filled circle too.

Or double-circle, triangle, or open sesame.

Color

Default color: match the text.

Explicit red: red marks matter here.

Custom blue: blue dots beneath blue text.

Position: over / under, left / right

Default over right: marked above.

Below: marked under.

Over left: left side.

Skip punctuation

Default: "hello, world!"

Skip: "hello, world!"

Custom string

Star marker: special words shine.

Vertical writing-mode

縦書きで 傍点 がついた文字列の例。

傍点は 右側 に表示されます。
