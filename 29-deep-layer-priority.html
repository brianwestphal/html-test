@layer priority + !important reversal

Normal cascade: later layer wins. With !important: earlier layer wins — order reversed. Unlayered normal wins over all layers; unlayered !important LOSES to layered !important.

layered: base < theme < overrides → green
unlayered → orange (beats all layers)

!important declarations (earlier layer wins)

!important: base wins → yellow

Cascade origin × !important matrix (per spec)

Origin × layer × importance is ordered, from lowest to highest precedence:

  1. Animations (transition / @keyframes — out of scope, no animation here)
  2. UA layered/unlayered normal
  3. User layered/unlayered normal
  4. Author layered (low) → unlayered → layered (high) — for normal
  5. Inline style normal
  6. Author layered (high) → unlayered → layered (low) — for !important (reversed)
  7. Inline style !important
  8. User !important
  9. UA !important
  10. Transition !important — N/A here
