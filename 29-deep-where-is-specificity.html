:where / :is / :not / :has & specificity counting

:where → 0 specificity

.a — text is BLUE (plain .a wins over :where(.high-spec, #my-id, [data-x]) .a which is 0)

:is → max specificity of args

div.b — text is RED (:is(#non-existent, .b) is 1,0,0 ≫ div.b 0,1,1)

:not → max specificity of args

.c — text is RED (:not(.skip, #foo) .c is 1,0,1 ≫ div .c 0,1,1)

:has → max specificity of args

.d with span#hot inside — yellow bg
.d without — white bg

Class repetition trick (.case.case.case)

.case .one (single .case) — yellow bg

nested .case.case → still 0,2,1, but rule .case.case.case .one matches → red

Attribute vs class — same level, later wins

later wins — green from .row.tag (defined after [data-x="x"])

Universal selector ignored for specificity

* .uni vs .uni — same 0,1,0 → later wins (red)

Inline (style attribute) > any selector

inline beats any selector (green)
